# Using Images in Blog Posts

You can add images to your posts in two ways:

1. Using regular markdown syntax:
![Go Gopher](/static/images/gopher.png)

2. Using HTML for more control:
<img src="/static/images/gopher.png" alt="Go Gopher" style="max-width: 300px; border-radius: 8px;">

Images can be:
- PNG, JPG, GIF formats
- Placed in the static/images directory
- Referenced using relative paths
